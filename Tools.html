This time around we decided not to concentrate too much on the website as we wanted to perfect our content as much as possible because we found we spent too much time in A2 perfecting the webpage. So only towards the end of our project, we started to use GitHub to apply all our content in the correct files and therefore the audit trail on the Git repository does not really reflect how much work each team member applied to the project solely by looking at the GitHub repository alone. We communicated quite a lot through teams in our group chat named “Drive Aware A3- Main Chat”. Here you will find many files uploaded and shared documents in which we edited all our work here, so everyone could see the progress. In saying this, we had a few hiccups where some team members would download the file and then work on it on their desktop not realising someone else was working on it, in turn creating confusion between some team members. But was quickly resolved without too much time lost. Overall great teamwork and fantastic team members.
