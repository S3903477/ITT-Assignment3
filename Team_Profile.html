Group Processes


Our group worked well together in Assignment 2 considering we had just met each other, therefore we did not make too many changes for Assignment 3. The meeting times and days as well as the agendas, recordings & notes were carried over from Assignment 2.

One area that we thought could use improvement was keeping up with each other’s progress. We thought if we had just one shared document as our content version then it would be easier for us all to know where each other was at and chip in if needed. In Assignment 2, there was some confusion near the end with some improvement text not being included due to not having the latest version.

For Assignment 3, we thought we would try working off one group document. The idea was that we would then only have one version and everyone would know where the others were up to. Unfortunately, this did not work as planned. The document was difficult to format and work with in Microsoft Teams. It was easier to use our own version of Word. So, we started downloading our own version with the intention of copying back the final version. The end result was many different versions of files and versions being proof read that were not truly the final version. It did not exactly go to plan. We also found there were a lot of files saved into our main chat in Teams. Lots of versions. It became a little confusing to know what was the latest.

We also were not able to delete old files for some reason. So possibly the reliance on sharing files in the chat repository was not the ideal way for us.
